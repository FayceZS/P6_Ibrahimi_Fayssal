Afin d'accéder à la base de données :

-Ajouter le fichier dbConnect.js au dossier middlewares

Pour lancer l'application :

I-Accéder au dossier frontend depuis le terminal puis :

1-Taper npm install

2-Taper npm start

II-Accéder au dossier backend depuis le terminal puis :

1-Taper npm install

2-Taper npm start

Dans le navigateur accéder à l'url : http://localhost:4200/
